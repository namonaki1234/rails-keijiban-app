# プロジェクト名
rails-keijiban-app

## 📌 概要
このプロジェクトは **掲示板を実現するためのツール/アプリケーション** です。  


## 🚀 特徴
- 🔹 **特徴1**：掲示板を簡単に実装
- 🔹 **特徴2**：複数のサーバーでコメントの視認が可能
- 🔹 **特徴3**：socket.ioを用いてクライアントとサーバーの連携を簡潔なコードで書いている



## 🛠️ インストール方法
### 🔹 前提条件
- Node.js 


### 🔹 セットアップ
```sh
git clone https://github.com/namonaki1234/rails-keijiban-app
cd rails-keijiban-app
npm install
